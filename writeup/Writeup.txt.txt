Amazon S3 is storage for the Internet. It can make web-scale computing easier for developers. 
AWS S3 provides a simple web services interface that can be used to store and retrieve any amount of data anytime anywhere on the web.
The container for objects stored in Amazon S3 is called an Amazon S3 bucket. 
Amazon S3 gives any developer access to the same infrastructure that Amazon uses to run its own global network of websites.
